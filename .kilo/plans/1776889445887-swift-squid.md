## Implementation Details

### 1.1 JWT Authentication
- Use `python-jose[cryptography]` for JWT signing
- HS256 algorithm with secure 256-bit secret key
- 15 minute access token expiration, 7 day refresh token
- Add `@authenticated` decorator for protected endpoints
- Validate token signature and expiration on every request

### 1.2 CORS Configuration
- Remove wildcard `*` origin
- Only allow explicitly configured origins from `.env`
- Disable `allow_credentials` unless strictly required
- Add `Vary: Origin` header

### 1.3 Security Headers
Implement these headers **on both backend and frontend**:
```
Strict-Transport-Security: max-age=63072000; includeSubDomains
X-Frame-Options: DENY
X-Content-Type-Options: nosniff
Content-Security-Policy: default-src 'self'; script-src 'self'; object-src 'none'
Referrer-Policy: strict-origin-when-cross-origin
Permissions-Policy: geolocation=(), microphone=(), camera=()
```

### 1.4 Rate Limiting
- Use `slowapi` with in-memory store (Redis for production)
- Limits:
  - 60 requests / minute per IP address
  - 10 requests / minute per authenticated user for /api/analyze
  - 5 failed login attempts lockout for 15 minutes

### 2.1 RBAC
- Define roles: `patient`, `doctor`, `admin`
- Add `@requires_role(role)` decorator
- Backend enforces role check for every protected endpoint
- Patient can only access their own records
- Doctor can access all patient records in their practice

### 2.3 Secure Cookies
- All auth tokens stored in `HttpOnly`, `Secure`, `SameSite=Strict` cookies
- No authentication data stored in localStorage
- Cookie domain restricted to exact origin
- Expiration matches token expiration

After implementation:
- [ ] All API endpoints return 401 without valid token
- [ ] CORS rejects requests from unknown origins
- [ ] Security headers are present on all responses
- [ ] Rate limiting blocks excessive requests
- [ ] Doctor cannot access admin endpoints
- [ ] Patient cannot access other patient data
- [ ] Cookies are HttpOnly and Secure
- [ ] Input with invalid ranges is rejected
- [ ] Docker runs as non-root user
